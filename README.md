# 🔧 نظام إدارة البلاغات — PWA

نظام ويب عربي كامل لإدارة بلاغات الصيانة، يعمل كـ **تطبيق أندرويد** بدون App Store.

---

## ⚡ تشغيل سريع

### ويندوز
```
انقر نقراً مزدوجاً على ملف: تشغيل_النظام.bat
```

### لينكس / ماك
```bash
pip3 install -r requirements.txt
python3 start.py
```

---

## 📱 تثبيت التطبيق على الموبايل (أندرويد)

### الخطوات:
1. **شغّل النظام** على الكمبيوتر/السيرفر
2. **افتح المتصفح Chrome** على الموبايل
3. **اكتب الرابط:**
   ```
   http://IP_الكمبيوتر:8080
   ```
   > مثال: `http://192.168.1.5:8080`
4. **سيظهر بانر أصفر** في أسفل الشاشة يقول **"تثبيت التطبيق"**
5. اضغط **📲 تثبيت** ← اضغط **إضافة**
6. ✅ **يظهر أيقونة التطبيق على الشاشة الرئيسية**

### ملاحظة - إذا لم يظهر البانر:
- افتح قائمة Chrome (⋮) ← **"إضافة إلى الشاشة الرئيسية"**

---

## 🔗 معرفة الـ IP تلقائياً

عند تشغيل `start.py` يظهر الرابط مباشرةً في الشاشة:
```
✅ النظام جاهز!

📱 رابط الدخول من الموبايل:
   http://192.168.1.5:8080

🖥️  رابط الدخول من الكمبيوتر:
   http://localhost:8080
```

## 🏗️ هيكل المشروع

```
maintenance_app/
├── تشغيل_النظام.bat    ← تشغيل بنقرة واحدة (ويندوز)
├── start.py            ← تشغيل الكل معاً
├── start.sh            ← تشغيل لينكس/ماك
├── app.py              ← نظام Streamlit
├── pwa_server.py       ← سيرفر FastAPI + PWA Proxy
├── requirements.txt
├── static/
│   ├── manifest.json       ← إعدادات PWA
│   ├── service-worker.js   ← عمل بدون إنترنت
│   ├── offline.html        ← صفحة انقطاع الإنترنت
│   └── icons/              ← أيقونات كل المقاسات
├── pages/              ← صفحات النظام
└── utils/              ← الأدوات والحسابات
```

## ⚙️ الإعدادات المتقدمة

### تغيير المنفذ
```bash
# تشغيل على منفذ مختلف
PWA_PORT=9000 python3 start.py
```

### HTTPS (للإنتاج - اختياري)
للحصول على رابط `https://` تحتاج:
1. نطاق (Domain)
2. شهادة SSL (مجانية من Let's Encrypt)
3. Nginx كـ Reverse Proxy
